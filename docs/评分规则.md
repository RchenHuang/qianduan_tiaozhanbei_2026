# NeuroFlex 认知训练评分规则

## 总体评分原则

### 评分范围
- 所有游戏统一使用 **0-100分** 评分制
- 60分为及格线，80分为优秀线，95分以上为卓越

### 评分构成
所有游戏的评分由三个核心维度组成：

1. **准确率权重 (70%)**：正确完成任务的比例
2. **速度权重 (20%)**：相对于标准时间的表现
3. **稳定性权重 (10%)**：反应时间的一致性

### 通用计算公式
```
总分 = 准确率分数 × 0.7 + 速度分数 × 0.2 + 稳定性分数 × 0.1
```

### 1. 舒尔特方格 (Schulte Grid) ✅ 已实现

#### 评分维度
- **准确率 (70%)**：正确点击数 / 总点击数
- **速度 (20%)**：基于标准完成时间
- **稳定性 (10%)**：反应时间方差

#### 标准时间设定（基于心理学研究数据）
| 网格大小 | 标准时间 | 优秀时间 | 卓越时间 | 参考来源 |
|---------|---------|---------|---------|---------|
| 3×3     | 5.4秒   | 4.0秒   | 3.0秒   | 估算值 |
| 4×4     | 9.6秒   | 7.0秒   | 5.5秒   | 估算值 |
| 5×5     | 20.0秒  | 12.0秒  | 8.0秒   | 研究数据* |

*参考数据来源：
- 成人（18岁以上）：8秒内为卓越，20秒为中等水平
- 青少年（12-14岁）：16秒内为优秀，26秒为中等水平
- 儿童（7-12岁）：26秒内为优秀，42秒为中等水平

#### 计算公式
```javascript
// 准确率分数 (0-100)
accuracyScore = (correctClicks / totalClicks) * 100

// 速度分数 (0-100)
speedRatio = standardTime / actualTime
speedScore = Math.min(100, speedRatio * 100)

// 稳定性分数 (0-100)
avgReactionTime = sum(reactionTimes) / reactionTimes.length
variance = sum((time - avgReactionTime)²) / reactionTimes.length
stabilityScore = Math.max(0, 100 - (variance / 1000))

// 总分
finalScore = accuracyScore * 0.7 + speedScore * 0.2 + stabilityScore * 0.1
```

### 2. Stroop 色词干扰训练 ✅ 已实现

#### 评分维度
- **准确率 (70%)**：正确判断颜色的比例
- **速度 (20%)**：平均反应时间表现
- **抗干扰能力 (10%)**：一致性试验与不一致试验的反应时间差

#### 标准反应时间（基于心理学研究数据）
| 难度级别 | 标准反应时间 | 优秀反应时间 | 卓越反应时间 | 参考来源 |
|---------|-------------|-------------|-------------|---------|
| 基础模式 | 800ms      | 600ms      | 450ms      | 研究数据* |
| 进阶模式 | 1000ms     | 750ms      | 600ms      | 研究数据* |

*参考数据来源：
- 成人Stroop测试平均反应时间：600-1000ms
- 不一致条件通常比一致条件慢100-200ms
- 优秀表现者可达到450-600ms范围

#### 计算公式
```javascript
// 准确率分数
accuracyScore = (correctResponses / totalResponses) * 100

// 速度分数
avgReactionTime = sum(reactionTimes) / reactionTimes.length
speedRatio = standardReactionTime / avgReactionTime
speedScore = Math.min(100, speedRatio * 100)

// 抗干扰分数
congruentAvg = average(congruentTrials)
incongruentAvg = average(incongruentTrials)
interferenceEffect = incongruentAvg - congruentAvg
interferenceScore = Math.max(0, 100 - (interferenceEffect / 10))

finalScore = accuracyScore * 0.7 + speedScore * 0.2 + interferenceScore * 0.1
```

### 3. 序列工作记忆训练 ✅ 已实现

#### 评分维度
- **准确率 (70%)**：正确回忆序列的比例
- **速度 (20%)**：记忆和回忆的效率
- **记忆广度 (10%)**：能够处理的序列长度

#### 标准表现（基于数字广度测试研究数据）
| 序列长度 | 标准准确率 | 优秀准确率 | 卓越准确率 | 参考来源 |
|---------|-----------|-----------|-----------|---------|
| 4个     | 85%       | 95%       | 100%      | 研究数据* |
| 6个     | 70%       | 85%       | 95%       | 研究数据* |
| 8个     | 55%       | 70%       | 85%       | 研究数据* |

*参考数据来源：
- 成人数字广度：正向6-7位数字，反向4-5位数字
- 8+位数字表示优异的工作记忆
- 大多数人在6-7个项目时达到记忆容量上限

#### 计算公式
```javascript
// 准确率分数
itemAccuracy = correctItems / totalItems
orderAccuracy = correctOrder ? 1 : 0
accuracyScore = (itemAccuracy * 0.6 + orderAccuracy * 0.4) * 100

// 速度分数 (基于记忆时间)
standardMemoryTime = itemCount * 1000 // 每项1秒
speedRatio = standardMemoryTime / actualMemoryTime
speedScore = Math.min(100, speedRatio * 100)

// 记忆广度分数
spanScore = Math.min(100, (itemCount / 8) * 100)

finalScore = accuracyScore * 0.7 + speedScore * 0.2 + spanScore * 0.1
```

- **准确率 (70%)**：正确识别目标声音的比例
- **抗噪能力 (20%)**：在不同信噪比下的表现
- **反应速度 (10%)**：识别和响应的速度

#### 信噪比标准（基于听觉注意力研究数据）
| 信噪比 | 标准准确率 | 优秀准确率 | 卓越准确率 | 参考来源 |
|-------|-----------|-----------|-----------|---------|
| 30%   | 60%       | 75%       | 85%       | 研究数据* |
| 20%   | 75%       | 85%       | 95%       | 研究数据* |
| 12%   | 90%       | 95%       | 100%      | 研究数据* |

*参考数据来源：
- 选择性听觉注意任务反应时间：400-800ms
- 信噪比越低，任务难度越高
- 正常成人在中等信噪比下准确率应达到75%以上

#### 计算公式
```javascript
// 准确率分数
digitAccuracy = correctDigits / totalDigits
orderAccuracy = correctOrder / totalSequences
accuracyScore = (digitAccuracy * 0.7 + orderAccuracy * 0.3) * 100

// 抗噪能力分数
noiseResistance = Math.min(100, (snr / 40) * 100)

// 反应速度分数 (基于输入完成时间)
standardInputTime = digitCount * 2000 // 每个数字2秒
speedRatio = standardInputTime / actualInputTime
speedScore = Math.min(100, speedRatio * 100)

finalScore = accuracyScore * 0.7 + noiseResistance * 0.2 + speedScore * 0.1
```

### 6. 规则导向分类训练 ✅ 已实现

#### 评分维度
- **分类准确率 (70%)**：正确分类的比例
- **规则理解速度 (20%)**：掌握新规则的速度
- **认知灵活性 (10%)**：规则切换时的适应能力

#### 标准表现（基于Wisconsin卡片分类测试研究数据）
| 维度数量 | 标准准确率 | 标准学习试次 | 优秀学习试次 | 参考来源 |
|---------|-----------|-------------|-------------|---------|
| 单维度   | 90%       | 8次         | 5次         | 研究数据* |
| 双维度   | 75%       | 15次        | 10次        | 研究数据* |

*参考数据来源：
- Wisconsin卡片分类测试：10岁儿童表现与成人相当
- 成人通常需要10-20次试验学习新规则
- 认知灵活性随年龄发展，成人期达到稳定水平

#### 计算公式
```javascript
// 分类准确率分数
accuracyScore = (correctClassifications / totalClassifications) * 100

// 规则理解速度分数
standardLearningTrials = dimensions === 1 ? 8 : 15
speedRatio = standardLearningTrials / actualLearningTrials
speedScore = Math.min(100, speedRatio * 100)

// 认知灵活性分数 (基于规则切换后的错误率)
switchErrorRate = switchErrors / switchTrials
flexibilityScore = Math.max(0, (1 - switchErrorRate) * 100)

finalScore = accuracyScore * 0.7 + speedScore * 0.2 + flexibilityScore * 0.1
```

## 评分等级划分

### 分数等级
- **🏆 卓越 (95-100分)**：认知能力极强，表现优异
- **⭐ 优秀 (80-94分)**：认知能力良好，有提升空间
- **✅ 良好 (60-79分)**：认知能力一般，需要持续训练
- **📈 待提升 (0-59分)**：认知能力较弱，建议加强练习

### 建议反馈
- **卓越**：保持当前水平，可尝试更高难度
- **优秀**：表现不错，继续保持训练频率
- **良好**：有进步空间，建议增加训练时间
- **待提升**：需要更多练习，建议降低难度开始
