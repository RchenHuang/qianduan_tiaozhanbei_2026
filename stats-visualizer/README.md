# 老年人健康监测系统

一个基于Vue 3的老年人健康指标可视化前端项目，用于实时监测和展示老年人的心率、血压、血糖等健康数据。

## 🌟 项目特点

- **现代化UI设计**：采用Element Plus组件库，界面美观专业
- **实时数据可视化**：使用ECharts实现丰富的图表展示
- **响应式布局**：适配不同屏幕尺寸，支持移动端访问
- **健康状态监控**：智能判断各项指标的健康状态
- **模拟数据演示**：内置JSON格式的演示数据，便于测试

## 📊 可视化指标

- **心率监测**：实时显示当前心率值和历史趋势
- **血压监测**：同时展示收缩压和舒张压数据
- **血糖监测**：跟踪血糖变化趋势
- **综合健康评分**：多维度健康状况评估

## 🎨 技术栈

- **Vue 3** - 渐进式JavaScript框架
- **Vite** - 现代化构建工具
- **Element Plus** - Vue 3组件库
- **ECharts** - 数据可视化图表库
- **vue-echarts** - Vue封装的ECharts组件

## 🎯 健康标准参考

### 心率 (bpm)
- 正常范围：60-100
- 偏低：< 60
- 偏高：> 100

### 血压 (mmHg)
- 正常范围：收缩压 90-140，舒张压 60-90
- 高血压：收缩压 > 140 或 舒张压 > 90
- 低血压：收缩压 < 90 或 舒张压 < 60

### 血糖 (mmol/L)
- 正常范围：3.9-7.0
- 偏低：< 3.9
- 偏高：> 7.0

## 🛠️ 开发指南

### 添加新的健康指标

1. 在 `healthData` 中添加新的数据结构
2. 创建对应的状态判断函数
3. 设计相应的图表配置
4. 在模板中添加展示组件

### 自定义样式

- 修改CSS变量调整主题色
- 在 `<style>` 标签中调整组件样式
- 使用Element Plus的主题定制功能

## 📝 数据格式

项目使用JSON格式的模拟数据，结构如下：

```json
{
  "heartbeat": {
    "current": 72,
    "history": [
      {"time": "08:00", "value": 68},
      {"time": "10:00", "value": 72}
    ]
  },
  "bloodPressure": {
    "systolic": 125,
    "diastolic": 80,
    "history": [
      {"time": "08:00", "systolic": 120, "diastolic": 78}
    ]
  },
  "glucose": {
    "current": 5.2,
    "history": [
      {"time": "08:00", "value": 5.1}
    ]
  }
}
```
